# Fitness Tracker Program Explanation
I have been asked to create a fitness tracker using Windows Forms that will allow users to log their activities for points.
The following information will need to be collected from the user:
+ Name: Take users name as string
+ Age: Take users age as int, Use TryParse to validate the age.
+ Exercise Type: Type of exercise completed.
+ Exercise Duration: The length of the exercise in minutes. Store as an int. Use TryParse to validate the duration.
+ Water: Use radio buttons to determine if the user drank water that day. Put radio button controls in a group box container.
+ Smoke: Use radio buttons to determine if the user smokes. Put radio button controls in a group box container.
